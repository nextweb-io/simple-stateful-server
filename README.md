simple-stateful-server
======================

A simple stateful server backed by [Nextweb](http://nextweb.io).

This project is an implementation of the
 [StatefulContext](https://github.com/mxro/server-api/blob/master/src/main/java/de/mxro/server/contexts/StatefulContext.java) 
 as defined in the [server-api](https://github.com/mxro/server-api) project.
 
Further resources:

- [Nextweb data model](http://nextweb.io/docs/nextweb-data-model.value.html)
- [Nextweb Java API](http://nextweb-io.github.com/nextweb-java/latest/)
- [Nextweb blog](http://nextweb-api.blogspot.com)
